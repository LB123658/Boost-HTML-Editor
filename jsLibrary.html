Boost HTML Editor







Boost HTML Editor now supports limited JavaScript functions including alerts, dark mode, and more. To use a JavaScript alert that says "Welcome", use the function myAlert(). To make a button reload your website, use the function reloadPage(). If you want to allow dark mode in your website use the function darkMode().

*IMPORTANT. These functions are stored in the Boost HTML Editor site so you can use them, however, for them to continue working on their own when your site is just an .html file you need to put the following code in for these functions to work. For the function myAlert() you need to put alert("Welcome!"); in the section for that function. For the reloadPage() function you need location.reload(); written in for it to work. For the darkMode() function you need to give the id "body" to the <body> tag and ad document.getElementById("body").style.background = "#282828";. Click here for full examples.
