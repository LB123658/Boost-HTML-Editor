Boost HTML Editor







Boost HTML Editor only allows HTML and CSS. PHP and JavaScript do not work on it. Special features include the wide screen mode so you can see what your website will actually look like, and an easy to read green color.

The editor was made with 57.8% HTML, 30% CSS, and 12.2% JavaScript. The most recent update now lets you download your code as an HTML file which then can be uploaded to make a website, making coding even easier with Boost HTML Editor. Remember to click the "Save changes" button before downloading though.

Newly updated: JS Library to easily style buttons. To use it, ad a script tag with src="https://lb123658.github.io/Boost-HTML-Editor/js/style-library.js" and the body tag with the onload="style()" function. Add the class red-default, green-default, or blue-default to style them with that color.
